# Customer Churn Prediction

## 📌 Project Overview
This project predicts whether a customer will churn (leave) or not using machine learning techniques.

## 🛠 Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn

## 📊 Dataset
Telco Customer Churn Dataset (Kaggle)

## 🤖 Model Used
- Logistic Regression
- Random Forest (optional)

## 📈 Results
- Achieved ~78% accuracy
- Identified key factors influencing churn

## 🔍 Key Insights
- Customers with higher charges are more likely to churn
- Long-term customers churn less
- Contract type impacts churn behavior

## 🚀 Future Improvements
- Hyperparameter tuning
- Advanced models (XGBoost)
- Deployment using Flask
